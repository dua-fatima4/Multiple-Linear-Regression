📊 Multiple Linear Regression Project
📌 Project Overview

This project implements Multiple Linear Regression from scratch using the Normal Equation method in Python to predict target values based on multiple independent variables.

📁 Dataset

The dataset was used to train and test the regression model.
The model was trained using feature matrix X and target variable y.

⚙️ Technologies Used

Python

NumPy

Matplotlib

Scikit-learn (for dataset splitting only)

📓 Algorithm Implementation

The regression model was implemented manually using the Normal Equation:

β = (XᵀX)⁻¹Xᵀy

This implementation does not rely on built-in LinearRegression from sklearn.

📈 Results

The model successfully predicts target values and demonstrates strong understanding of regression mathematics and matrix operations.
